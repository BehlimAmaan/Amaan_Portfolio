# ⚙ Installation

Clone the repository

```bash
git clone https://github.com/BehlimAmaan/Portfolio.git
```

Move into the project

```bash
cd Portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm start
```

or

```bash
npm run dev
```
